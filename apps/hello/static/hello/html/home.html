Project Description

It's a test project. The main technical requirements are as follows:

  • It should be a web application
  • The frontend should be built with AngularJS
  • UI Design - better use Google Material, build responsive UI (you may use with twitter-bootstrap)
  • The backend should be in Django
  • It should have client side and server side validation
  • It should work asynchronously
  • It should have automated tests for all functionality (both backend and frontend)
  • Should be deployed on getbarista.com
  • All heavy tasks should be run on Celery

Project Features

  • There should be a landing page that describes the project and the author
  • As a visitor I want to be able to register and authenticate with my Facebook account
  • I want to be able to add tasks
  • I want to be able to in-place update/delete tasks
  • I want to be able to prioritise tasks by dragging them up and down
  • I want to be able to choose deadline for my task
  • I want to be able to mark a task as 'done'
  • I want to be able to build a team by inviting my Facebook friends
  • If I'm invited to different teams I want to be able to switch between them
  • I want to be able to assign/re-assign tasks to my teammates
  • I want to be able to sort tasks by date, assignee, title
  • All my changes to tasks, including task title, assigness, done, due to, and priority should be immediately shown on every open page so that other users will see the changes in real time
